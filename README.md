> 💡 대구대학교 컴퓨터공학과에서 웹 개발과 AI/컴퓨터 비전을 깊이 있게 다루고 있습니다.

---

## 🧭 About

- 🎓 대구대학교 컴퓨터공학과 재학 중입니다.
- 🤖 임베디드 시스템(Jetson Nano, Raspberry Pi, Arduino)을 활용한 프로젝트 경험이 있습니다.
- 🔍 YOLOv5와 SSD-MobileNet-v2로 객체 탐지 및 컴퓨터 비전을 수행합니다.
- 🌐 Spring Boot와 React로 풀스택 개발을 경험했습니다.
- 🤝 오픈소스 프로젝트 협업과 Hugging Face 생태계를 활용했습니다.
- 📊 데이터 분석(R, MongoDB, MySQL)부터 웹 개발까지 기술 스펙트럼을 다룹니다.
